# 🧠 Claims Intelligence Platform  
### Built using Snowflake CoCo + Streamlit

---

## 🚀 Overview

This project was developed as part of a **Hackathon using Snowflake CoCo (Cortex + Copilot)** to demonstrate how modern data platforms can accelerate end-to-end analytics and AI-powered applications.

We built a **Claims Intelligence Platform** that transforms raw insurance data into a fully interactive analytics application with:

- 📊 Real-time dashboards
- 🧠 AI-powered chatbot (Cortex Analyst)
- 🏗️ Automated data modeling using prompts
- ⚡ End-to-end pipeline built entirely in Snowflake

---

## 🧩 What Makes This Special?

This solution is **fully powered by Snowflake CoCo**, showcasing:

- Prompt-driven **table generation**
- Automated **data pipelines**
- AI-assisted **star schema modeling**
- Semantic layer creation using **Cortex**
- Natural language analytics via **Cortex Analyst**
- Streamlit app deployed directly within Snowflake

---

## 🏗️ Architecture

Raw Data → Transformation Layer → Star Schema → Semantic Layer → Streamlit App + Cortex Chatbot


Everything from ingestion to UI is built **inside Snowflake ecosystem using CoCo capabilities**.

---

## 📊 Streamlit Application

The Streamlit app is the core of this project and provides:

### 🔹 Features

- 📈 Interactive dashboards (Claims, Financials, Risk Metrics)
- 📍 Region-wise and severity analysis
- 📊 Trend analysis & KPI tracking
- 🤖 AI Chatbot powered by Cortex Analyst
- 🧠 Natural language → SQL → Insights

### 🔹 Key Capabilities

- Uses **Snowflake Snowpark session**
- Integrated with **Cortex COMPLETE & Analyst APIs**
- Dynamic query generation using **semantic model**
- Clean UI with Plotly visualizations

👉 Streamlit Code: https://github.com/hy-kipi/Kipi-Cocothon-Claim2Fame/blob/main/Streamlit_App_Chatbot.py
